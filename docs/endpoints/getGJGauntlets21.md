# getGJGauntlets21.php

Gets the gauntlet levels.

## Parameters

### Required Parameters

**secret** - Wmfd2893gb7

### Optional Parameters

**gameVersion** - 21

**binaryVersion** - 35

**gdw** - 0

**special** - 1 (fetches the 2.2 gauntlets)

## Response

A list of [gauntlet objects](/docs/resources/server/gauntlet.md), separated by a pipe `|`.

## Example

<!-- tabs:start -->

### **Python**

```py
import requests

data = {
    "secret": "Wmfd2893gb7"
}

req = requests.post("http://boomlings.com/database/getGJGauntlets21.php", data=data)
print(req.text)
```
